# AI Workshop Web Presentation

عرض ويب عربي RTL لورشة الذكاء الاصطناعي، مع شاشة مقدم، شاشة جمهور، إدارة صور/فيديو، ولوجو اختياري.

## التشغيل المحلي

```bash
npm install
npm run dev
```

افتح:

```text
http://localhost:5173
```

في التشغيل المحلي تظهر الصفحة الرئيسية كلوحة مقدم. شاشة الجمهور النظيفة:

```text
http://localhost:5173/?audience=1
```

لوحة المقدم مع أدوات الصور واللوقو:

```text
http://localhost:5173/?presenter=1&admin=1
```

## اختصارات العرض

- `Space` أو `ArrowRight`: التالي.
- `ArrowLeft`: السابق.
- `A`: فتح شاشة الجمهور.
- `B`: تعتيم/إظهار شاشة الجمهور.
- `F`: ملء الشاشة.
- `O`: عرض كل الشرائح.
- `M`: تبديل الوضع المختصر/الكامل.
- `+` و `-`: تكبير/تصغير ملاحظات المتحدث.

## إضافة صور أو فيديو

الطريقة الأسهل والدائمة:

1. ضع الصورة أو الفيديو داخل `public/media`.
2. استخدم اسمًا مطابقًا للشريحة والمكان:

```text
public/media/slide-09-main.jpg
public/media/slide-09-thumb-1.jpg
public/media/slide-10-main.mp4
```

الصيغ المدعومة: `jpg`, `jpeg`, `png`, `webp`, `gif`, `mp4`, `webm`.

## استخدام مدير الوسائط

افتح:

```text
http://localhost:5173/?presenter=1&admin=1
```

ثم اضغط `إدارة الصور والفيديو` واختر الشريحة والمكان. إذا كانت الصورة مقصوصة غيّر `Fit` إلى `contain`.

## إضافة اللوقو

الطريقة الدائمة:

```text
public/media/logo.png
```

أو افتح لوحة المقدم مع `admin=1` ثم استخدم زر `إعداد اللوقو`. اللوقو يظهر في العرض حسب الإعدادات، ولا تظهر أدوات الإدارة للجمهور.

## Docker المحلي

```bash
docker compose -f docker-compose.local.yml up -d --build
```

ثم افتح:

```text
http://127.0.0.1:4561
```
